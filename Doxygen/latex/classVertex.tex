Finds the nearest adjacent edge to this vertex. 

This function iterates through all the adjacent edges and finds the one with the minimum weight.

\begin{DoxyReturn}{Returns}
Edge$<$\+T$>$$\ast$ A pointer to the edge with the minimum weight adjacent to this vertex. 
\end{DoxyReturn}
